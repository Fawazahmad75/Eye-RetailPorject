# Eye-RetailPorject
Designed and built a computer vision based retail inventory system to track incoming stock and monitor real-time shelf availability, significantly reducing manual auditing time.
Developed a full-stack web application using Next.js, creating a live dashboard with automated restock alerts and inventory insights.
Integrated Roboflow object detection models to accurately identify products and track shelf positioning in real time.
Leveraged Antigravity to vibe-code and rapidly prototype features.
